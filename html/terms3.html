Working directory

Generells sätt, är den nyaste versionen "working directory", alltså där man jobbar just nu. varje fil man jobbar på, är här. Git upptäcker automatiskt ändringar.

Staging

Ett sätt att testa ändringar innan de läggs till i den faktiska GitHub Enterprise instansen.

Head

En speciell version av en gren, oftast den senaste commiten vid grenens spets.

Revert

När man Revertar en Pull request på GitHub, så öppnas automatiskt en ny pull begäran, och den har en commit som Revertar merge commiten från den orginella merge begäran. I Git kan man gå tillbaks med Git Revert.

Reset

Om man vill återställa det man jobbar på; alltså återställa till den senaste committen, så kan man använda en reset.
